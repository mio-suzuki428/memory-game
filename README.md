# memory-game
神経衰弱アプリ/ランキング機能なし版
JavaScriptで作成した神経衰弱ゲームです。

## 機能

* プレイヤー名入力
* 神経衰弱ゲーム
* ローカルランキング TOP10
* プレイヤー名の自動保存

## 使用技術

* HTML
* CSS
* JavaScript
* LocalStorage

## プレイ

以下のURLからプレイできます。

https://Mio-Suzuki428.github.io/memory-game/

## 開発経緯

当初はJava Servletを利用した動的Webプロジェクトとして開発し、オンラインランキング機能の実装を行っていました。

GitHub Pagesで公開するため、ローカルランキング機能へ変更し、静的Webアプリとして再構築しました。
